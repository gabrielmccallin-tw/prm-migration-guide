{% comment %} The care card include displays content in a highlighted way. Parameters: - include.content: Required. The content of what you want to be shown highlighted. HTML by default. - include.heading: The heading. HTML by default. - include.accessibility_text: The text that describes what information is presented for accessbility. Default is 'Information' - include.markdown: Set to true if your include.content is in Markdown, so that it is transformed. {% endcomment %}
